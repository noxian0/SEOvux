# Changelog

## 0.1.0 — First public release

### Included

- Native Windows desktop app with light and dark themes.
- Live, browser-rendered SEO audits using Chromium for desktop and mobile page checks.
- Redirect, status, metadata, canonical, robots, indexability, on-page, mobile, accessibility, performance, and trust checks.
- Evidence-backed findings, healthy signals, screenshots, CSV/JSON/PDF exports, and local audit history.
- Same-domain public site crawl and content extraction with robots.txt enforcement, rate limiting, duplicate-signal detection, and CSV/JSON exports.
- SSRF protection and clear handling for blocked pages, timeouts, login walls, and rendering limitations.

### Important limits

- SEOvux does not guarantee rankings, indexing, traffic, or business results.
- Site Crawl only processes publicly accessible same-domain HTML pages and respects robots.txt; it does not bypass logins, CAPTCHAs, paywalls, or access controls.
- The first release is for Windows and stores audit history on the local device.
